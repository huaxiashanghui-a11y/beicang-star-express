## ⚠️ 注意事项

1. **后端API**：当前部署的是前端，后端需要在本地运行 `npm run dev`
2. **数据存储**：当前使用本地 JSON 文件存储数据
3. **云端部署**：如需完整云端部署，需要配置云数据库

---

## 🚀 本地开发

```bash
# 启动前端
cd d:\beicang.js
npm run dev

# 启动后端（另一个终端）
cd d:\beicang.js\server
npm run dev
```
